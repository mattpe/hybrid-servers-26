# Hybridisovellukset-kurssin esimerkkipalvelimet

Laitettu kaikki sovellukset samaan repoon yksinkertaisuuden vuoksi (ei paras tapa).

Kloonattu Ilkan repot, jwa paketti tarvitsee päivityksen joka sovelluksessa: `npm upgrade jwa`.

Muutettu hybrid-types paketin lähde lokaaliksi palvelimien package.jsoneissa: `npm i ../hybrid-types`, muista ajaa `npm i` hybrid-types-repossa ensin.
